# 更新日志

## [1.3.0] - 2024-12-13

### 新增
- 🔒 **隐私模式**: 支持标记标题为隐私，隐私模式下自动隐藏敏感内容
- ↕️ **拖拽排序**: 支持拖拽标题及其下方内容整体移动
- 👁 隐私标记按钮，位于标题左侧
- 👁 阅读模式切换按钮，位于工具栏

### 优化
- 🛠️ 周报生成时自动过滤隐私内容
- 🛠️ H1/H2/H3 标题按删除键直接删除（不再降级）
- 🛠️ 导入待办时不再添加来源标题
- 🛠️ 运行端口改为 3001

### 技术
- 📦 新增 @dnd-kit 拖拽库支持
